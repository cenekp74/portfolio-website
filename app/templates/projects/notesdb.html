About this project

NotesDB was created to help students at my high school share study materials in a structured and organized way. My hope was that it will catch on and people will share their school work, eventually creating a database that would cover most subjects from the entire curriculum.
NotesDB is my first attempt at creating a full stack web app. Reflecting on it now, there are many aspects that I would do differently.

Key features

  • advanced search features - filters, sorting etc.
  • user system - with login, profile pictures, password reseting, ...
  • email system - automatic sending of emails from @notesdb.cz
  • view counter
  • light/dark theme

Current status and future plans

Due to low user activity, NotesDB was archived and moved from notesdb.cz to notesdb.potuznik.dev. In its current state, new user registration and email functionalities are disabled.

In the future, I would like to create a better version of this website - more user friendly, more features and better design.
