I am running my own server at home, hosting multiple services for my personal use.

These services include:

  • Jellyfin
  • Audiobookshelf
  • TriliumNext
  • Immich

The server that these services are hosted on is an old hp machine running ubuntu server 22.04. I use nginx as reverse proxy and docker for containing the services. To hide my public ip, I use cloudflare's proxy.

In the future, I would like to fully abandon big tech companies such as google and microsoft and replace their products with open source, self-hosted alternatives. I am currently in the proces of setting up my own mail server and nextcloud.
