About this project

Rust-labeler is a simple app made with tauri in rust. It provides a simple way to label images with a category. It features keyboard shortcuts for quick workflow.

Key features

  • images visualization - images are represented by dots for simple navigation and overview
  • keyboard shortcuts - keybindings for each category as well as for navigation
  • output as json - output is in json format for easy processing

Current status and future plans

I am using this app frequently myself, therefore I will keep maintaining it and adding new features. If you have any feature request, feel free to contact me via email.
